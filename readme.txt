如何使用
--------
执行 source envsetup.sh 设置环境变量


如何构建 toolchain
-------------------
使用 crosstool-ng-1.28.0 工具构建，使用的配置在 configs 目录下。
注意：需要 meson 和 ninja 的构建工具，并且在构建 ct-ng 之前就要安装好。（sudo apt install meson ninja-build）
