如何使用
--------
执行 source envsetup.sh 设置环境变量


如何编译出 riscv32-picolibc-elf gcc
-----------------------------------
使用 crosstool-ng-1.26.0 工具，samples 下的 riscv32-picolibc-elf 配置进行构建即可（建议去掉 multilib 和 c++ 的支持）
